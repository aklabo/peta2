# peta2

- A simple application with Node.js + express + ejs.
- Any kind of contribution will be welcomed by anyone in this organization.

#Getting Started

```
$ git clone git@github.com:aklabo/peta2.git
$ cd peta2/app
$ ./run-server.sh
```
